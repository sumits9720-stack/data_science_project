Crypto Trading Behavior & Market Sentiment Analysis

Analyzed 211k crypto trades against the Fear & Greed Index to reveal how sentiment drives risk. Found 'Fear' yields 60% larger positions & higher win rates, while 'Greed' drives volume but lowers efficiency. Built with Python & Power BI.
